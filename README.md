# PDFViewer Sample

This sample demonstrate how to implement the PDFViewer on a WPF app using the MVVM pattern.

## Project structure
```
PDFViewerWPFDemo/
	PDFViewerWPFDemo/
		Properties/
		Resources/						- App resources
		ViewModel/						- MVVM files
		App.config
		App.xaml
		App.xaml.cs
		MainWindow.xaml
		MainWindow.xaml.cs
		PDFViewerWPFDemo.csproj
	PDFViewerWPFDemo.sln
```
